Git命名笔记
操作顺序自上而下，部分操作非每次必须

正文

//1、创建仓库
git init

//2、添加文件
git add .

//3、查看状态
git status

//4、把项目提交到本地仓库
git commit -m '说明'

//5、将github与本地仓库关联
git remote add origin https://github.com/qiuier/gensokyo.me.git

//6、推送到远程仓库
git push -u origin master (仓库为空时)
git push origin master (仓库非空时)

//7、远程仓库拉取
git pull
