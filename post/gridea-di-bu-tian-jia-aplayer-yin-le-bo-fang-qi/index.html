Gridea底部添加Aplayer音乐播放器

思路

添加APlayer需要在Gridea的主题文件夹下修改和添加代码。

添加库引用

首先找到themes文件夹当前使用的主题，我使用的是Lemon主题。在主题文件夹下的head.ejs中添加以下代码。

  • link部分
<link rel="stylesheet" href="https:\\cdn.jsdelivr.net\npm\aplayer@1.10.1\dist\APlayer.min.css">
  • js部分
<script src="https:\\cdn.jsdelivr.net\npm\color-thief-don@2.0.2\src\color-thief.js"><\script>
<script src="https:\\cdn.jsdelivr.net\npm\meting@1.1.0\dist\Meting.min.js"><\script>
<script src="https:\\cdn.jsdelivr.net\npm\aplayer@1.10.1\dist\APlayer.min.js"><\script>

添加容器div

在footer.ejs中添加以下代码

<div id="aplayer"><\div>

使用js设置参数

在div下面添加

<script>
  const ap = new APlayer({
      container: document.getElementById('aplayer'),
      fixed: true,
      loop: 'none',
      autoplay: false,
      audio: [{
          name: '爆炒猫头鹰-Choco Strawberry（Hea2t\Star酱 Remix）',
          artist: 'Hea2t,Star酱',
          url: 'http:\\music.163.com\song\media\outer\url?id=1347291491.mp3',
          cover: 'http:\\p2.music.126.net\JRX5apIl5qViACGmAeTz6g==\109951163877718520.jpg?param=300x300',
          lrc: '[00:00.00] 暂无歌词',
      }]
  });
<\script>

其中参数意义

  • fixed: true开启底吸模式
  • loop: 'none'关闭循环播放
  • autoplay: false关闭自动播放
